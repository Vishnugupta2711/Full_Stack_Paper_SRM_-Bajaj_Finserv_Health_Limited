# 🚀 BFHL Hierarchy Analyzer  
**SRM Full Stack Engineering Challenge – Bajaj Finserv Health Limited**

A full-stack application that processes directed node relationships, constructs hierarchical trees, detects cycles, and provides structured insights via a REST API and interactive frontend.

---

## 📌 Live Links

- 🌐 Frontend: https://full-stack-paper-srm-bajaj-finserv.vercel.app/
- 🔗 Backend API: https://bfhl-backendvg.onrender.com/

---

## 🧠 Features

- ✅ Parses and validates node relationships (`A->B`)
- 🌳 Constructs multiple independent trees (forest)
- 🔁 Detects cycles in graph components
- 🚫 Identifies invalid inputs
- 🔁 Handles duplicate edges correctly
- ⚡ Resolves multi-parent conflicts (first-parent wins rule)
- 📊 Computes:
  - Tree depth
  - Total trees
  - Total cycles
  - Largest tree root (with tie-breaking)
- 🎯 Clean and interactive frontend UI
